# 📊 PDF to Excel | EasyUseHub

**PDF to Excel** is a free online tool to instantly convert your PDF tables into Excel (.xlsx) files. Upload your PDF, process it entirely in your browser, and download the Excel file—no software installation or API required.

## Features
- ✅ Convert PDF tables to Excel (.xlsx) in your browser
- ✅ Client-side processing for privacy and speed
- ✅ Multi-page PDF support (each page becomes a separate sheet)
- ✅ Max file size: 30MB
- ✅ Premium UI with clean design
- ✅ Ad placeholders for monetization
- ✅ Mobile-friendly & responsive layout
- ✅ Only essential buttons: All PDF Tools & Home

## How to Use
1. Click **Upload PDF** and select your PDF file (max 30MB).  
2. Click **Convert to Excel**.  
3. Once conversion completes, click **📥 Download Excel**.
